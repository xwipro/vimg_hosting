# v图床

## 1. 关于开发

- 前端: vite + vue3 + typescript
- 后端: php + mysql

## 2. 关于部署

- 前端:

> 需要安装 `nodejs` 的环境，之后在前端的目录下执行 `npm install` 安装必须得依赖，之后执行 `npm run build` 打包，打包完成之后会生成 `dist` 文件夹，将 `dist` 文件夹中的内容复制到你的服务器根目录下即可\[ 记住这里不要 `dist` 这个文件夹，只要里面的文件 \]。

- 后端:

> 将 `api` 文件夹中的sql文件导入到服务器中的mysql数据库中，之后将 `api` 文件夹整体丢到前端的根目录下即可。

## 3. 关于使用

- 前端: 访问你的服务器地址即可，默认端口是 `80`，如果需要修改端口，可以在 `vite.config.ts` 文件中修改。

- 后端: 访问你的服务器地址加上 `/api` 即可，关于php的配置可以在 `api/config.php` 文件中修改。

## 4. 关于其他

- 本项目仅供学习参考，禁止用于商业用途。
